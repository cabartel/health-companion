# Health Companion

A comprehensive health tracking skill for Heyron agents.

## Quick Start

```bash
/health start "Vitamin D" --benchmarks fatigue,brain-fog --weeks 4
/health med add "Metformin XR" --dosage 500mg --frequency daily
/health migraine add --severity 8 --triggers "stress"
/health score 6
/health benchmarks
```

## Features

- **Intervention Tracker** — Run controlled trials with custom benchmarks
- **Medication Tracking** — Keep a running list of medications and dosages
- **Migraine Tracking** — Log episodes with triggers, severity, and relief
- **Weekly Scores** — Track condition-specific scores over time
